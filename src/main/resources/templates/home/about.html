Acerca de Teafind

Teafind es un proyecto presentado como trabajo de fin de grado por Alejandro Asensio Álvarez, Jorge Asensio Álvarez y Cristian Chaparro Arteaga para el curso de Técnico Superior de Aplicaciones Web del instituto IES Rey Fernando VI.
Datos técnicos:

La web está desplegada y alojada en Google Cloud para permitir el acceso público.

Se ha configurado un dominio exclusivo para el acceso fácil desde una dirección web amigable y reconocible.

Teafind es un proyecto maven enteramente desarrollado en Java con Springboot, Hibernate y Thymeleaf para la generación de las plantillas de las vistas.

Se ha utilizado Bootstrap 4 en toda la web para dar formato y estilo, personalizado mediante CSS.

Los efectos o funcionalidades extra han sido desarrollados en JavaScript y jQuery.

Descripción del proyecto:

Teafind es una web de publicación y búsqueda de proyectos de software cuyo objetivo principal es facilitar la comunicación entre aquellas personas que compartan un mismo objetivo.

Es el nexo de unión entre una idea o un proyecto ya iniciado y el talento con el que, de otro modo, sería difícil converger.
